# Decla portal

Decla portal is a React front-end for Decla Tecnologia.

## Installation

Use the package manager [GIT](https://github.com/jmncruz/portal-decla.git) to install Decla portal.

```bash
git fetch
git pull
npm install
```

## Usage

```bash
npm run dev

# got to 'browser'
http://127.0.0.1:3000


```
